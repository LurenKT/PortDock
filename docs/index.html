Cascade start级联启动

One click starts a service and everything it depends on, in dependency order: database first, then API, then the app. Commands are saved locally as plain JSON. 点一下,按依赖顺序拉起整条链:先数据库,再 API,最后应用。启动命令保存在本地一个 JSON 文件里。

02

Why trust a process monitor?凭什么信任一个进程监控器?

It's fully open source and tiny — ~3k lines of Swift you can read in an afternoon. Everything stays on your machine. 它完全开源,而且很小 —— 约 3000 行 Swift,一个下午就能读完。一切都留在你的机器上。
  • Only reads the system's own lsof / ps output 只读系统自带的 lsof / ps 输出
  • No telemetry, no network egress无遥测,不向外发任何数据
